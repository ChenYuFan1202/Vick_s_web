# Vick's Personal Website

> 范振宇 (Vick) 的個人網站 - 清華大學服務科學研究所碩士生

## 網站簡介

這是一個展示個人經歷、作品集與故事的靜態網站，採用響應式設計，支援深色/淺色模式切換。

## 網站結構

```
├── index.html          # 首頁
├── about me.html       # 關於我
├── portfolio.html      # 作品集
├── stories.html        # 故事集
├── contact.html        # 聯絡資訊
└── assets/
    ├── css/            # 樣式表
    ├── js/             # JavaScript
    ├── font/           # 字型
    └── img/            # 圖片資源
```
